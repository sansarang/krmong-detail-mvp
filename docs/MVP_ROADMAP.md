# PageAI MVP 로드맵 (채널·전환·패키징)

**제품 포지션 한 줄:** 글만 생성하는 도구가 아니라, **「이 글이 어디서·어떻게 팔리게 할지」**와 **「규제·과장 리스크를 줄이며 팔리게 할지」**까지 묶어 주는 쪽으로 이동한다.

이 문서는 **2주 안에 끝낼 수 있는 단위**로 후보를 쪼개고, **지금 추천하는 1순위 MVP**와 그 다음 후순위를 정리한다.

---

## 1. 프레임: 어디 / 어떻게 / 무엇을 줌

| 축 | 의미 (사용자에게 주는 가치) | 예시 산출물 |
|----|----------------------------|-------------|
| **어디 (채널)** | 플랫폼별 검색·피드·구매 동선이 다름 | 채널별 한 줄 전략, 포맷별 체크리스트 |
| **어떻게 (전환)** | 클릭·저장·구매까지 이어지는 문장/배치 | 훅 문장 2안, CTA 문구, 썸네일 카피 힌트 |
| **무엇을 줌 (패키징)** | 복붙 단위로 끊어서 주면 실행이 빨라짐 | HTML + 캡션 + 메타 + (추후) 이미지 스펙 |

**이미 코드에 있는 것:** 주문 결과 화면의 **발행 전 안전 점검**(과장·효능 휴리스틱) → 「망할 일 없이」 축의 얇은 1차.

**이번에 추가한 것:** 같은 화면의 **채널 발행 키트**(플랫폼별 전략·전환·패키징 가이드 + 훅 복사) → 「어디·어떻게」 축의 v0.

**추가 반영 (P1·P2·P4 v1):** **전환 A/B 카피**(제목·첫문장 3안, 복사 + 1번 섹션에 적용), **메타/OG 패키지**(HTML·JSON 복사), **업종 추정 + 추가 규칙**(건강·뷰티·식품 등) — 코드 기준 1차 구현.

## 2. 후보 MVP 슬라이스 (기능 단위 × 타깃)

| 우선순위 | 슬라이스 | 타깃 고객 | 핵심 가치 | 상태 |
|----------|----------|-----------|-----------|------|
| **P0** | **채널 발행 키트 v1** | 스마트스토어·쿠팡 파트너 블로거, 1인 셀러 | 발행 직전에 채널별로 무엇을 할지 고정 | **출시** |
| P1 | 전환 A/B 카피 (제목·첫문장 3안) + 복사·적용 | 트래픽은 있으나 클릭이 약한 사용자 | 같은 본문으로 상단만 갈아 끼우기 | **v1 출시** (템플릿 기반) |
| P2 | 메타/OG·검색 스니펫 패키지 export | SEO 의식 있는 셀러 | 블로그 외 사이트·링크 공유까지 | **v1 출시** |
| P3 | 팀/승인 (읽기 전용 링크 + 코멘트) | 소규모 팀 | 리스크 문구 수정을 사람이 확인 | **v0** (`ORDER_SHARE_SECRET` 서명 링크 + 공유 페이지; 인라인 코멘트 저장은 후속) |
| P4 | 규칙 엔진 고도화 (업종별 금칙어 세트) | 건강·뷰티·식품 등 민감 카테고리 | 오탐·미탐 줄이기 | **v1 출시** (휴리스틱 확장, 지속 개선) |

## 3. 추천: **2주 MVP = 채널 발행 키트 v1** (단일 선택)

**왜 이걸 1개로 고르나**

- 이미 주문 결과에 **플랫폼 선택·HTML/캡션 복사**가 있음 → 같은 진입점에서 **「왜 이 포맷인지 + 무엇을 더 챙기면 팔리는지」**만 얹어도 포지션이 바뀜.
- LLM 추가 호출 없이도 **템플릿 + 제품 맥락**으로 v0 제공 가능 (빠름).
- 「규제·과장」 점검과 **한 화면에서 쌍**으로 보여 주면 스토리가 완성됨: *안전하게 + 어디에 어떻게.*

**완료 정의 (Done)**

- [x] 플랫폼별 **전략 1문단 + 전환 팁 + 패키징 체크** 문구가 **ko/en/ja/zh**로 표시된다.
- [x] **훅 문장 2안**을 버튼으로 복사할 수 있다.
- [x] (코드) `docs/MVP_ROADMAP.md`에 후순위와 타깃이 정리되어 있다.

**2주 쪼개기 (역할 나눌 때)**

| 주 | 초점 |
|----|------|
| **1주** | 플랫폼 매핑(네이버·티스토리·WP·인스타·브런치·기타 HTML) 문구 확정, UI 카드, 복사 UX, i18n |
| **2주** | 훅 템플릿을 섹션 제목과 연동, 사용자 인터뷰 3명으로 문구 톤 수정, 랜딩/온보딩 한 문장 반영 |

**성공 지표 (초기)**

- 주문 결과 화면에서 **키트 영역 클릭/복사** 비율
- 이후 설문/인터뷰: *「글만 쓰는 것보다 도움이 됐나?」* 5점 척도
